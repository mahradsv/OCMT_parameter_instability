This zip file contains codes and data for the replication of Monte Carlo simulations and empirical results in "Variable Selection in High Dimensional Linear Regressions with Parameter Instability", version July 2024, by Alexander Chudik, M. Hashem Pesaran, and Mahrad Sharifvaghefi.

This zip file contains the following two folders:

1. MC simulations
2. applications

Each of these folders is self-contained, and each contains its own ReadMe.txt file that explains how to run the codes.
