This zip file contains codes and data for the replication of empirical results in "Variable Selection in High Dimensional Linear Regressions with Parameter Instability", version July 2024, by Alexander Chudik, M. Hashem Pesaran, and Mahrad Sharifvaghefi.

This zip file contains the following three folders:

1. Forecasting monthly rate of stock price changes
2. Forecasting quarterly output growth (33 countries)
3. Forecasting Euro Area quarterly output growth

These folders correspond to the three applications presented in the paper. Each of these folders is self-contained, and each contains its own readme.txt file that explains how to run the codes.
